# HANK Model Estimation for Chile

Replication and adaptation of the HANK-X model from **Bayer, Born, and Luetticke (2024, AER)** — *"Shocks, Frictions, and Inequality in US Business Cycles"* — estimated for Chile as an emerging economy application.

The replication builds on the authors' **BASEforHANK Julia toolbox**, with a modified model to account for data availability constraints in Chile (no tax progressivity series). Income and wealth distribution parameters are calibrated using series from the **World Inequality Database (WID)** and the **Encuesta Financiera de Hogares (EFH)**.

## Getting started

**1. Download the BASEforHANK toolbox** (v1.5.0, required once):
```bash
julia setup_BASEHANK.jl
```
This clones `src/`, `examples/`, and `docs/` from the [BASEforHANK GitHub repo](https://github.com/BASEforHANK/BASEtoolbox.jl) at the pinned version. These directories are gitignored and not bundled with this repo.

**2. Install Julia dependencies** (exact versions pinned in `Manifest.toml`):
```bash
julia --project=. -e "using Pkg; Pkg.instantiate()"
```

**3. Install R dependencies:**
```r
renv::restore()
```

**4. Obtain raw data** (see [Data sources](#data-sources) below — not included in the repo).

**5. Run the data pipeline:**
```bash
Rscript code/cleaning.R
```

## Key modifications from BBL baseline

| Parameter | BBL (US) | Chile |
|---|---|---|
| Top 10% wealth share | WID (US) | WID Chile |
| Transition to entrepreneur | Top 10% wealth share | WID Chile |
| Portfolio liquidity | Kaplan, Violante & Weidner (2014) / SCF | EFH 2024 |
| Discount factor | Capital-to-Output (National Accounts) | Chilean National Accounts |
| Tax rate | National Accounts | Chilean National Accounts |
| Tax progressivity | Ferriere-Navarro methodology | **Not available — simplified** |
| Labor share | Average labor income share | Chilean National Accounts |

## Data sources

Raw data is **not included** in this repository. Place files in the paths below before running the pipeline.

| Dataset | Access | How to obtain | Local path |
|---|---|---|---|
| BASEforHANK Toolbox v1.5.0 | Public | Run `julia setup_BASEHANK.jl` (downloads automatically) | `src/`, `examples/` |
| World Inequality Database (WID) | Public | Download from [wid.world](https://wid.world): country = CHL, variables = wealth shares (top 10%, top 1%), years = 1990–2023 | `data/raw/WID/` |
| Encuesta Financiera de Hogares (EFH) 2024 | Public | Download from [Banco Central de Chile](https://www.bcentral.cl/areas/encuestas-economicas/encuesta-financiera-de-hogares): select EFH 2024, download all `.dta` files | `data/raw/EFH/` |
